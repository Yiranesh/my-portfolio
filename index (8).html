Project 03

Weather App

I originally built this as a terminal app, then converted it into a web application using Flask. It was a good way to learn how routing and APIs work in a real project.

  • Gets live weather data using OpenWeatherMap API
  • Shows temperature, humidity, wind speed, and conditions
  • Includes a 5-day forecast with daily cards
  • Compares weather side-by-side between two cities
  • Deployed live — accessible from any browser

What I learned: This introduced me to APIs, Flask routing, Jinja2 templates, and how to deploy a real web app so anyone can use it online.
